## 🚀 Proyectos destacados

### LiveSpot

Plataforma frontend para eventos, experiencias en vivo y reservas, construida como proyecto de portafolio orientado a producto digital.

Incluye catálogo de eventos, filtros, detalle dinámico, formulario de reserva, persistencia local con `localStorage`, dashboard básico y componentes reutilizables.

**Stack:** React, Vite, Tailwind CSS, React Router, JavaScript

**Repositorio:**  
https://github.com/josecorrea01/livespot

---

### SkillTrack-Empresas

Aplicación web para gestión de empresas, colaboradores y cursos de capacitación.

Incluye estructura MVC, vistas JSP, persistencia con JPA/Hibernate, base de datos H2/MySQL y despliegue en Railway.

**Stack:** Java, Spring Boot, JSP, JPA/Hibernate, MySQL, H2, Bootstrap, Railway

**Repositorio:**  
https://github.com/josecorrea01/skilltrack-empresas

**Deploy:**  
https://skilltrack-empresas-production.up.railway.app

---

### theLook Commerce BI Case

Caso práctico de **Data Modeling & Business Intelligence** desarrollado sobre datos de e-commerce usando BigQuery.

Incluye consultas SQL analíticas, staging enriquecido, marts de negocio, métricas comerciales y capa semántica conceptual.

**Stack:** BigQuery, SQL, Data Modeling, Business Intelligence

**Repositorio:**  
https://github.com/josecorrea01/thelook-commerce-bi-case

---

### Academic Migration API

API REST backend para estructuración y migración de datos académicos hacia MongoDB.

Incluye arquitectura base con Express, conexión a MongoDB Atlas, validaciones con Zod y documentación técnica del MVP.

**Stack:** Node.js, Express, MongoDB Atlas, Mongoose, Zod

**Repositorio:**  
https://github.com/josecorrea01/academic-migration-api
